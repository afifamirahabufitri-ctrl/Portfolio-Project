
## Classroom Performance Dataset
A collection of practical datasets and exercises for learning data analytics, data cleaning, and business intelligence.

#**Project Overview**

This project demonstrates end-to-end SQL analysis using a simulated classroom database. The objective is to analyze student demographics, academic performance, and club participation while showcasing fundamental and advanced SQL concepts commonly used by Data Analysts.

The project covers:

- Database creation
- Table relationships
- Data exploration
- Joins
- Aggregations
- CASE statements
- Common Table Expressions (CTEs)
- Window Functions
- Subqueries
- Business-focused analytical questions
