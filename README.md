
## Classroom Performance Dataset
A collection of practical datasets and exercises for learning data analytics, data cleaning, and business intelligence.

#**Project Overview**

This project demonstrates end-to-end SQL analysis using a simulated classroom database. The objective is to analyze student demographics, academic performance, and club participation while showcasing fundamental and advanced SQL concepts commonly used by Data Analysts.

The project covers:

- Database creation
- Table relationships
- Data exploration
- Joins
- Aggregations
- CASE statements
- Common Table Expressions (CTEs)
- Window Functions
- Subqueries
- Business-focused analytical questions


#**Analysis Outcome**

1. **Top Performing Student in Each Grade**

This query uses a Common Table Expression (CTE) and Window Function (RANK()) to calculate each student's total score across subjects and rank students within their respective grade levels. The final result returns the highest-performing student(s) from each grade, enabling grade-level performance benchmarking and recognition of top achievers.

**Business Case Description**

School administrators would like to identify the best-performing student in each grade based on overall academic performance. This analysis aggregates subject scores at the student level and ranks students within each grade, allowing educators to recognize top achievers and compare performance across grade levels.
